We created a Book Exchange System that gives book lovers and book readers a place to post a book with a comment and/or a rating. The user is also able to send and receive messages via the Inbox Tab. Give the Book System a try and review your favorite books!

We are a group of Computer Science Undergraduate students utilizing our software engineering technical skills to create this system. We used our knowledge of Django, Bootstrap, Databases, and Python to create a functional system for book users.
